# Getting Started

## Prerequisites

- Node.js
- npm or yarn

## Installation

### Install Dependencies

```bash
npm install
# or
yarn install
```

### Run Server

```bash
npm dev:server
# or
yarn dev:server
```

### Run Client

```bash
npm dev:client
# or
yarn dev:client
```
